# Slide `<ru-slide>`

Is a gereric component for slide show case.

## Example Usage

Note: For `boolean` attributes, `true` means simply setting the attribute (e.g. `<ru-yourcomponent prop>`) and `false` means not setting the attribute (e.g. `<ru-yourcomponent>` without `prop` attribute).

### Standard

```html
<ru-slide open>
  <!-- Content start -->
  <ru-button label="label" raised></ft-button>
  <!-- ... -->
  <!-- Content end -->
</ft-slide>
```
