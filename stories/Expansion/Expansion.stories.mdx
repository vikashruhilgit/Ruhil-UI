# Expansion `<ru-expansion>`

Provide an expandable details-summary view.

## Example Usage

Note: For `boolean` attributes, `true` means simply setting the attribute (e.g. `<ru-yourcomponent prop>`) and `false` means not setting the attribute (e.g. `<ru-yourcomponent>` without `prop` attribute).

### Basic

<Canvas withToolbar>
  <Story id="ruhil-ui-component-expansion--basic" height="50px" />
</Canvas>

```html
<ru-expansion>
  <span slot="title">Title</span>
  <span slot="description">Description</span>
  <p>Here's some content</p>
</ft-expansion>
```

### Groups

<Canvas withToolbar>
  <Story id="ruhil-ui-component-expansion--groups" height="50px" />
</Canvas>

```html
<ru-expansion name="group">
  <span slot="title">Expansion 1</span>
  <span slot="description">Description</span>
  <p>Here's some content</p>
</ft-expansion>

<ru-expansion name="group">
  <span slot="title">Expansion 2</span>
  <span slot="description">Description</span>
  <p>Here's some content</p>
</ft-expansion>

<ru-expansion name="group">
  <span slot="title">Expansion 3</span>
  <span slot="description">Description</span>
  <p>Here's some content</p>
</ft-expansion>
```

### Disabled

<Canvas withToolbar>
  <Story id="ruhil-ui-component-expansion--disabled" height="50px" />
</Canvas>

```html
<wl-expansion name="group" disabled>
  <span slot="title">Disabled Expansion</span>
  <span slot="description">Description</span>
  <p>Here's some content</p>
</wl-expansion>
```
