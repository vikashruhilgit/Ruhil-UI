# Formfield `<ft-formfield>`

<p>
  A form field is a text caption for ft input elements including
  <code>&lt;ft-checkbox&gt;</code>
  <code>&lt;ft-radio&gt;</code>, and
  <code>&lt;ft-switch&gt;</code>.
</p>
<p>
  It is equivalent to the native
  <code>&lt;label&gt;</code>
  element in that it forwards user interaction events to the input element. For
  example, tapping on the label causes the associated input element to toggle
  and focus.
</p>

## Example Usage

Note: For `boolean` attributes, `true` means simply setting the attribute (e.g. `<ft-yourcomponent prop>`) and `false` means not setting the attribute (e.g. `<ft-yourcomponent>` without `prop` attribute).

### With checkbox

<Canvas withToolbar>
  <Story
    id="firstech-ui-form-component-formfield--with-checkbox"
    height="100px"
  />
</Canvas>

```html
<ft-formfield label="Tomato">
  <ft-checkbox checked></ft-checkbox>
</ft-formfield>
```

### nowrap label with checkbox

<Canvas withToolbar>
  <Story
    id="firstech-ui-form-component-formfield--nowrap-label-with-checkbox"
    height="100px"
  />
</Canvas>

```html
<style>
  ft-formfield[nowrap] {
    width: 150px;
  }
</style>
<ft-formfield label="really really long label" nowrap>
  <ft-checkbox></ft-checkbox>
</ft-formfield>
```

## Properties/Attributes

<table>
  <thead>
    <tr>
      <th>Name</th>

  </thead>
  <tbody>
    <tr>
      <td>
        <code>label</code>
      </td>
      <td>
        <code>string</code>
      </td>
      <td>
        The text to display for the label and sets a11y label on input.

      <td>

      </td>
      <td>

    </tr>
    <tr>
      <td>
        <code>spaceBetween</code>
      </td>
      <td>
        <code>boolean</code>
      </td>
      <td>

      </td>
    </tr>
    <tr>
      <td>
        <code>nowrap</code>
      </td>
      <td>
        <code>boolean</code>
      </td>

    </tr>
