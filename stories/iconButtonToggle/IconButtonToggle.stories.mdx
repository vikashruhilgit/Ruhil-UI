# Icon Button Toggle `<ft-icon-button-toggle>`

<p>
  Toggle buttons can be used to group related options. To emphasize groups of
  related toggle buttons, a group should share a common container.
</p>

<p>
  Icons can be used as toggle buttons when they allow selection, or deselection,
  of a single choice, such as marking an item as a favorite.
</p>

## Example Usage

Note: For `boolean` attributes, `true` means simply setting the attribute (e.g. `<ft-yourcomponent prop>`) and `false` means not setting the attribute (e.g. `<ft-yourcomponent>` without `prop` attribute).

### Standard

`on = true` and `on = false`

```html
<ft-icon-button-toggle
  onIcon="sentiment_very_satisfied"
  offIcon="sentiment_very_dissatisfied"
></ft-icon-button-toggle>
```

### Disabled

```html
<ft-icon-button-toggle
  disabled
  onIcon="sentiment_very_satisfied"
  offIcon="sentiment_very_dissatisfied"
></ft-icon-button-toggle>
```

### With toggled aria label

```html
<ft-icon-button-toggle
  ariaLabelOn="Very satisfied"
  ariaLabelOff="Very dissatisfied"
  onIcon="sentiment_very_satisfied"
  offIcon="sentiment_very_dissatisfied"
></ft-icon-button-toggle>
```

## Custom Style

Most users should include the following in their application HTML when using icons:

```html
ft-icon-button-toggle { color: tomato; }
```

## Fonts

Most users should include the following in their application HTML when using icons:

```html
<link
  href="https://fonts.googleapis.com/css?family=Material+Icons&display=block"
  rel="stylesheet"
/>
```

<p>
  This loads the <em>Material Icons</em> font, which is required to render
  icons, and is
  <em>not</em> loaded automatically. If you see plain text instead of an icon, then
  the most likely cause is that the Material Icons font is not loaded.
</p>

<p>
  To see all icons that are available in the Material Icons font, see
  <a href="https://material.io/resources/icons/" rel="nofollow">
    Material Icons
  </a>.
</p>

<p>
  For technical details about the Material Icons font, see the
  <a href="https://google.github.io/material-design-icons/" rel="nofollow">
    Material Icons Developer Guide
  </a>.
</p>
