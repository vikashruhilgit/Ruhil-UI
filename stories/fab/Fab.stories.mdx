# Floating Action Button `<ft-fab>`

<p>A floating action button (FAB) represents the primary action of a screen.</p>

## Example Usage

Note: For `boolean` attributes, `true` means simply setting the attribute (e.g. `<ft-yourcomponent prop>`) and `false` means not setting the attribute (e.g. `<ft-yourcomponent>` without `prop` attribute).

### Usage

<Canvas withToolbar>
  <Story
    id="ruhil-ui-component-fab-floating-action-button--fab"
    height="100px"
  />
</Canvas>

### Standard

```html
<ft-fab icon="edit"></ft-fab>
```

### Standard - slotted

```html
<ft-fab label="edit">
  <ft-icon slot="icon">edit</ft-icon>
</ft-fab>
```

### Mini

```html
<ft-fab mini icon="add"></ft-fab>
```

### Extended

```html
<ft-fab extended icon="shopping_cart" label="Add to cart"></ft-fab>
```

### Customize colors

```html
ft-fab { --mdc-theme-secondary: white; --mdc-theme-on-secondary: black; }
```

## Fonts

Most users should include the following in their application HTML when using icons:

```html
<link
  href="https://fonts.googleapis.com/css?family=Material+Icons&display=block"
  rel="stylesheet"
/>
```

<p>
  This loads the <em>Material Icons</em> font, which is required to render
  icons, and is
  <em>not</em> loaded automatically. If you see plain text instead of an icon, then
  the most likely cause is that the Material Icons font is not loaded.
</p>

<p>
  To see all icons that are available in the Material Icons font, see
  <a href="https://material.io/resources/icons/" rel="nofollow">
    Material Icons
  </a>.
</p>

<p>
  For technical details about the Material Icons font, see the
  <a href="https://google.github.io/material-design-icons/" rel="nofollow">
    Material Icons Developer Guide
  </a>.
</p>

    <tr>
      <td>
        <code>icon</code>
      </td>

    </tr>
    <tr>
      <td>
        <code>label</code>
      </td>
      <td>
        <code>string</code>
      </td>
      <td>
        The label to display when using the <code>extended</code> layout, and
        the <code>aria-label</code> attribute in all layouts.
      </td>
    </tr>
    <tr>
      <td>
        <code>mini</code>
      </td>
      <td>
        <code>boolean</code>
      </td>
      <td>
        Modifies the FAB to be a smaller size, for use on smaller screens.
        Defaults to <code>false</code>.
      </td>
    </tr>
    <tr>
      <td>
        <code>reducedTouchTarget</code>
      </td>
      <td>
        <code>boolean</code>
      </td>
      <td>
        Sets the minimum touch target of the default-sized mini fab to
        recommended 48x48px.
      </td>
    </tr>
    <tr>
      <td>
        <code>extended</code>
      </td>
      <td>
        <code>boolean</code>
      </td>
      <td>
        Enable the <em>extended</em> layout which includes a text label.
        Defaults to <code>false</code>.
      </td>
    </tr>
    <tr>
      <td>
        <code>showIconAtEnd</code>
      </td>
      <td>
        <code>boolean</code>
      </td>
      <td>
        When in the <em>extended</em> layout, position the icon after the label,
        instead of before. Defaults to <code>false</code>.
      </td>
    </tr>
  </tbody>
</table>
