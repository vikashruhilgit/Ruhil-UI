# Circular Progress `<ft-circular-progress>`

<p>
  Progress indicators express an unspecified wait time or display the length of
  a process.
</p>
<p>
  The four color circular progress indicator rotates between four colors in
  indeterminate mode.
</p>

## Example Usage

Note: For `boolean` attributes, `true` means simply setting the attribute (e.g. `<ft-yourcomponent prop>`) and `false` means not setting the attribute (e.g. `<ft-yourcomponent>` without `prop` attribute).

### Determinate

<Canvas withToolbar>
  <Story
    id="firstech-ui-component-circular-progress--determinate"
    height="100px"
  />
</Canvas>

```html
<ft-circular-progress-four-color
  progress="0.7"
></ft-circular-progress-four-color>
```

### Indeterminate

<Canvas withToolbar>
  <Story
    id="firstech-ui-component-circular-progress--indeterminate"
    height="100px"
  />
</Canvas>

```html
<ft-circular-progress-four-color
  indeterminate
></ft-circular-progress-four-color>
```

### Styled

<p>
  The four-color Circular Progress Indicator does not specify the value of the
  four colors by default. These colors may be specified as follows:
</p>

<Canvas withToolbar>
  <Story id="firstech-ui-component-circular-progress--styled" height="100px" />
</Canvas>

```html
<style>
  ft-circular-progress-four-color {
    --mdc-circular-progress-bar-color-1: #2196f3;
    --mdc-circular-progress-bar-color-2: #e91e63;
    --mdc-circular-progress-bar-color-3: #ffc107;
    --mdc-circular-progress-bar-color-4: #03dac5;
  }
</style>
<ft-circular-progress-four-color
  indeterminate
></ft-circular-progress-four-color>
```
