# Button `<ru-button>`

Buttons allow users to take actions, and make choices, with a single tap.

## Example Usage

Note: For `boolean` attributes, `true` means simply setting the attribute (e.g. `<ru-yourcomponent prop>`) and `false` means not setting the attribute (e.g. `<ru-yourcomponent>` without `prop` attribute).

### Standard

<Canvas withToolbar>
  <Story
    id="ruhil-ui-component-button--button&args=raised:true"
    height="100px"
  />
</Canvas>

```html
<ru-button icon="code" label="Hello i'm a Button" raised=""></ft-button>
```

    </tr>
    <tr>
      <td>
        <code>trailingIcon</code>
      </td>
      <td>
        Icon to show <em>after</em> the label. Overrides{' '}
        <code>trailingIcon</code> property. Use <code>label</code> or the{' '}
        <code>trailingIcon</code> property to set the <code>aria-label</code>.
      </td>
    </tr>
    <tr>
      <td>
        <em>default</em>
      </td>
      <td>
        Default content to display between both icons and after label.{' '}
        <strong>NOTE:</strong> It is highly recommended to set the{' '}
        <code>label</code> property instead of projecting text as it will also
        set the <code>aria-label</code>
      </td>
    </tr>
  </tbody>
</table>
