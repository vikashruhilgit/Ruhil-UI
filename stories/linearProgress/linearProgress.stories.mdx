# Linear Progress `<ft-linear-progress>`

Progress indicators express an unspecified wait time or display the length of a process.

## Example Usage

Note: For `boolean` attributes, `true` means simply setting the attribute (e.g. `<ft-yourcomponent prop>`) and `false` means not setting the attribute (e.g. `<ft-yourcomponent>` without `prop` attribute).

### Determinate

<Canvas withToolbar>
  <Story
    id="firstech-ui-component-linear-progress--determinate"
    height="100px"
  />
</Canvas>

```html
<ft-linear-progress progress="0.5"></ft-linear-progress>
```

      </td>

      </td>
    </tr>

<p>
  Default value for <code>--mdc-linear-progress-buffering-dots-image</code>:
</p>

```js
data:image/svg+xml,%3Csvg version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink' x='0px' y='0px' enable-background='new 0 0 5 2' xml:space='preserve' viewBox='0 0 5 2' preserveAspectRatio='none slice'%3E%3Ccircle cx='1' cy='1' r='1' fill='%23e6e6e6'/%3E%3C/svg%3E
```

<p>
  If you paste that data url into your browser you can see and inspect the SVG.
  You may also notice that it is simply a colored circle with some transparent
  space to its right. If you would like to change the color of the circle, you
  can simply change the <code>fill="%23e6e6e6"</code> to any valid color
  property (n.b. <code>%23</code> is the url-encoded equivalent of{' '}
  <code>#</code>).
</p>
