# Snackbar `<ft-nackbar>`

<p>
  Snackbars provide brief messages about app processes at the bottom of the
  screen.
</p>

## Example Usage

Note: For `boolean` attributes, `true` means simply setting the attribute (e.g. `<ft-yourcomponent prop>`) and `false` means not setting the attribute (e.g. `<ft-yourcomponent>` without `prop` attribute).

### Standard

<Canvas withToolbar>
  <Story id="firstech-ui-component-snackbar--default" height="100px" />
</Canvas>

```html
<ft-snackbar
  id="photoErrorSnackbar"
  labelText="Can't send photo. Retry in 5 seconds."
>
</ft-snackbar>
```

### Optional action and dismiss button

<Canvas withToolbar>
  <Story
    id="firstech-ui-component-snackbar--optional-action-and-dismiss-button"
    height="100px"
  />
</Canvas>

```html
<ft-snackbar labelText="Can't send photo. Retry in 5 seconds.">
  <ft-button slot="action">RETRY</ft-button>
  <ft-icon-button icon="close" slot="dismiss"></ft-icon-button>
</ft-snackbar>
```

### Custom action button color

<Canvas withToolbar>
  <Story
    id="firstech-ui-component-snackbar--custom-action-button-color"
    height="100px"
  />

```css
ft-snackbar {
  --mdc-snackbar-action-color: #64dc17;
}
```

### Variants

#### Stacked

<p>
  Action buttons with long text should be positioned below the label instead of
  alongside it. Set the <code>stacked</code> attribute or property to enable
  this layout.
</p>

#### Leading

<p>
  By default, snackbars are centered horizontally within the viewport. On larger
  screens, they can optionally be displayed on the leading edge of the screen
  (the left side in LTR, or the right side in RTL). Set the <code>leading</code>
  attribute or property to enable this layout.
</p>
