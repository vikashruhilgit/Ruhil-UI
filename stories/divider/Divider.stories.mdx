# Divider `<ru-divider>`

Thin line that groups content in lists and layouts.

## Example Usage

Note: For `boolean` attributes, `true` means simply setting the attribute (e.g. `<ru-yourcomponent prop>`) and `false` means not setting the attribute (e.g. `<ru-yourcomponent>` without `prop` attribute).

### Basic

<Canvas withToolbar>
  <Story id="ruhil-ui-component-divider--divider" height="50px" />
</Canvas>

```html
<ru-divider></ft-divider>
```

### Vertical

<Canvas withToolbar>
  <Story id="ruhil-ui-component-divider--vertical" height="50px" />
</Canvas>

```html
<ru-divider vertical></ft-divider>
```

### Styled

```html
<style>
  ft-divider {
    --divider-color: red;
    --divider-size: 1px;
    height: 100px;
  }
</style>
<ru-divider></ft-divider>
```
