# Checkbox `<ft-checkbox>`

<p>
  Radio buttons allow the user to select one option from a set. Use radio
  buttons when the user needs to see all available options.
</p>

## Example Usage

Note: For `boolean` attributes, `true` means simply setting the attribute (e.g. `<ft-yourcomponent prop>`) and `false` means not setting the attribute (e.g. `<ft-yourcomponent>` without `prop` attribute).

### Standard

<Canvas withToolbar>
  <Story id="ruhil-ui-form-component-radio--standard" height="100px" />
</Canvas>

```html
<ft-radio name="myGroup" value="value1"></ft-radio>
<ft-radio name="myGroup" value="value2" checked></ft-radio>
```

### Custom color

<Canvas withToolbar>
  <Story id="ruhil-ui-form-component-radio--custom-color" height="100px" />
</Canvas>

```html
<style>
  ft-radio {
    background-color: #363636;

    --mdc-theme-secondary: #ff2929;
    --mdc-radio-unchecked-color: white;
  }
</style>

<ft-radio name="myGroup" checked></ft-radio>
<ft-radio name="myGroup"></ft-radio>
```

### Disabled custom color

<Canvas withToolbar>
  <Story
    id="ruhil-ui-form-component-radio--disabled-custom-color"
    height="100px"
  />
</Canvas>

```html
<style>
  ft-radio {
    background-color: #363636;

    --mdc-radio-disabled-color: #adadad;
  }
</style>

<ft-radio disabled name="myGroup"></ft-radio>
<ft-radio disabled name="myGroup" checked></ft-radio>
```

### With Form Field

<Canvas withToolbar>
  <Story id="ruhil-ui-form-component-radio--with-form-field" height="100px" />
</Canvas>

```html
<style>
  ft-formfield {
    display: block;
  }
</style>

<ft-formfield label="Home">
  <ft-radio name="location"></ft-radio>
</ft-formfield>

<ft-formfield label="Work">
  <ft-radio name="location"></ft-radio>
</ft-formfield>
```

          native <code>&lt;input&gt;</code>
