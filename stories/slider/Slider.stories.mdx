# Slider `<ft-slider>`

<p>Sliders allow users to make selections from a range of values.</p>

## Example Usage

Note: For `boolean` attributes, `true` means simply setting the attribute (e.g. `<ft-yourcomponent prop>`) and `false` means not setting the attribute (e.g. `<ft-yourcomponent>` without `prop` attribute).

### Continuous

```html
<ft-slider value="25" min="10" max="50"></ft-slider>
```

### Discrete

```html
<ft-slider discrete withTickMarks step="5" max="50" value="10"> </ft-slider>
```

### Methods

<table>
  <thead>
    <tr>
      <th>Name</th>
      <th>Description</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>
        <code>valueToValueIndicatorTransform(value: number) =&gt; string</code>
      </td>
      <td>
        Override this method to transform the value of a given knob to the text
        that should be displayed in the value indicator of a discrete slider.
        Additionally, this will set the <code>aria-valuetext</code>.
      </td>
    </tr>
    <tr>
      <td>
        <code>layout(skipUpdateUI = false) =&gt; Promise&lt;void&gt;</code>
      </td>
      <td>
        Recomputes the dimensions and re-lays out the component. This should be
        called if the dimensions of the slider itself or any of its parent
        elements change programmatically (it is called automatically on
        pointerdown). The skipUpdateUI parameter determines whether the UI
        should be updated alongside the internal clientRect model.
      </td>
    </tr>
