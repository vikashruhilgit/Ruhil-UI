# Switch `<ft-switch>`

<p>
  Switches toggle the state of a single setting on or off. They are the
  preferred way to adjust settings on mobile.
</p>

## Example Usage

Note: For `boolean` attributes, `true` means simply setting the attribute (e.g. `<ft-yourcomponent prop>`) and `false` means not setting the attribute (e.g. `<ft-yourcomponent>` without `prop` attribute).

### Standard

<Canvas withToolbar>
  <Story id="firstech-ui-form-component-switch--standard" height="100px" />
</Canvas>

```html
<ft-switch></ft-switch>
```

### Selected by default

<Canvas withToolbar>
  <Story
    id="firstech-ui-form-component-switch--selected-by-default"
    height="100px"
  />

<Canvas withToolbar>
  <Story id="firstech-ui-form-component-switch--disabled" height="100px" />

### With Form Field

<p>
  Most applications should use
  <code>&lt;ft-formfield&gt;</code>
  to associate an interactive label with the switch.
</p>

### CSS Custom Properties

<p>
  The switch may be customized using the <code>theme()</code> mixin to emit
  custom properties. The <code>theme()</code> mixin can be used for simple and
  complex theming, such as density.
</p>

```css
mwc-switch {
  --mdc-switch-selected-handle-color: #00897b;
  --mdc-switch-selected-track-color: #4db6ac;
}

mwc-switch.dense {
  --mdc-switch-state-layer-size: 44px;
}
```
