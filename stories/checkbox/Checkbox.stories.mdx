# Checkbox `<ft-checkbox>`

<p>
  Checkboxes allow the user to select one or more items from a set. Checkboxes
  can be used to turn an option on or off.
</p>

## Example Usage

Note: For `boolean` attributes, `true` means simply setting the attribute (e.g. `<ft-yourcomponent prop>`) and `false` means not setting the attribute (e.g. `<ft-yourcomponent>` without `prop` attribute).

### Standard

<Canvas withToolbar>
  <Story id="ruhil-ui-form-component-checkbox--standard" height="100px" />
</Canvas>

```html
<ft-checkbox checked></ft-checkbox>
<script>
  const checkbox = document.body.querySelector('ft-checkbox');
  checkbox.addEventListener('change', () => {
    console.log(`checkbox changed to ${checkbox.checked}`);
  });
</script>
```
